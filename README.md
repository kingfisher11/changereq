# Change of Request
 
 Sistem Change of request bagi urusan permohonan perubahan berkaitan sistem
 
 Clone
 ``` 
 git clone url
 ```
 
 In terminal, navigate to directory and run
 
```
composer install
``` 
 ```
 php artisan key:generate
 ```
```
php artisan migrate:fresh --seed
```

Run localhost
```
php artisan serve
```
